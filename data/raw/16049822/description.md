# QuRE Data Description

The dataset contains 2,111 unique requirements in natural, unconstrained English. Find details in the corresponding paper.

* **id**: A unique number identifying the data point.
* **requirement**: The requirement text as it is represented in the original specification.
* **defect**: A human-labeled assessment of whether the corresponding weak word in the requirement constitutes a quality defect (value `defect`) or not (`ok`).
* **weak_word**: A label representing a weak word apparent in the requirement.
